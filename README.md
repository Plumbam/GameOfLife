# GameOfLife

🧬 Игра «Жизнь» Конвея (B3/S23) - десктопное приложение для моделирования клеточного автомата.

## 📋 Описание

Проект реализует классические правила **B3/S23** (Birth 3 / Survival 2, 3) и предоставляет удобный интерактивный интерфейс для исследования эмерджентного поведения сложных систем.

**Приложение разработано в рамках учебной практики (ВВГУ, 2026)**

## ✨ Возможности

- **Интерактивный редактор**: рисование и удаление клеток мышью с поддержкой drag-and-drop
- **Гибкое управление**: запуск/пауза, пошаговое выполнение, очистка поля
- **Контроль скорости**: регулировка интервала между поколениями (10-500 мс)
- **Мониторинг**: отображение поколения и количества живых клеток
- **Детекция стабильности**: автоматическая остановка при статичном состоянии
- **Работа с файлами**: сохранение и загрузка в формате JSON
- **Тороидальное поле**: границы замыкаются сами на себя
- **Тёмная тема**: для снижения нагрузки на глаза

## 🛠 Технологический стек

- **Язык**: Python 3
- **GUI**: PySide6 (Qt for Python)
- **Вычисления**: NumPy (векторизованные операции)
- **Архитектура**: MVC (Model-View-Controller)
