Игра «Жизнь» Конвея (B3/S23)
Десктопное приложение для моделирования клеточного автомата «Жизнь», созданного математиком Джоном Конвеем. Проект реализует классические правила B3/S23 (Birth 3 / Survival 2, 3) и предоставляет удобный интерактивный интерфейс для исследования эмерджентного поведения сложных систем.
✨ Возможности
Интерактивный редактор: Рисование и удаление клеток мышью с поддержкой перетаскивания (drag-and-drop)
Гибкое управление: Запуск/пауза, пошаговое выполнение, очистка поля и генерация случайного состояния
Контроль скорости: Плавная регулировка интервала между поколениями (от 10 до 500 мс)
Мониторинг: Отображение номера текущего поколения и количества живых клеток в реальном времени
Детекция стабильности: Автоматическая остановка симуляции при достижении статичного состояния
Работа с файлами: Сохранение и загрузка конфигураций в удобочитаемом формате JSON (по умолчанию в ~/Documents/GameOfLife)
Тороидальное поле: Границы поля замыкаются сами на себя (мир на поверхности тора)
Современный UI: Интуитивно понятный интерфейс с принудительной тёмной темой для снижения нагрузки на глаза
🛠 Технологический стек
Язык: Python 3
GUI: PySide6 (Qt for Python)
Вычисления: NumPy (векторизованные операции для высокой производительности)
Архитектура: MVC (Model-View-Controller)

⚙️ Технические особенности
Оптимизация вычислений: Подсчет соседей и обновление поля реализованы без использования медленных циклов for в Python. Применяются векторизованные операции NumPy и функция np.roll() для сдвига массивов по всем 8 направлениям.
Производительность: Ядро симуляции обрабатывает поле 50×70 (3500 клеток) менее чем за 1 мс, что позволяет легко поддерживать частоту обновления свыше 100 кадров в секунду.
Масштабируемость: Архитектура готова к дальнейшему развитию, включая переход на OpenGL для рендеринга полей сверхбольшого размера.
